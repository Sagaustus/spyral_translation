Call for Human-in-the-Loop Translators

We are recruiting Yoruba-speaking scholars, students, and community members to review AI-generated interface translations. Your role is to approve, revise, or reject translations so that the final interface is accurate, culturally appropriate, and consistent.

The workflow is designed to make review safe and efficient: placeholders and HTML tags are protected, and automated QA checks flag risky strings (missing placeholders, tag mismatches, unbalanced braces, etc.).

What reviewers do

  • Check meaning and tone for Yoruba interface text.
  • Ensure placeholders remain intact (e.g., %s, {name}).
  • Ensure basic HTML tags remain intact where used (e.g., <b>, <a>).
  • Resolve terminology consistency (same UI concept → same Yoruba term).

How approval works (in this site)

  1. AI drafts land in machine_draft.
  2. Reviewers edit reviewer_text and set status to In review / Flagged.
  3. A superadmin approves final text into approved_text (status Approved).

Access is currently staff-only via Django admin.

How to join

Submit an application to become a reviewer. Once approved by a SuperAdmin, you will be able to review AI drafts for the language you selected.
