What happens

1
Protect placeholders

ICU tokens ({0}, %(name)s), HTML tags, and entities are replaced with numbered sentinels before translation so the engine cannot corrupt them.

2
Translate → machine draft

The selected engine produces a target-language draft. Sentinels are swapped back after translation.

NLLB-200 OpenAI Ollama
3
Back-translate → English

The draft is translated back to English using NLLB-200 (always local) to produce a comparable surface for scoring.

4
XLM-R similarity score

Cosine similarity between the original English source and the back-translated English. Score < 0.75 → low_similarity QA flag.

XLM-R sentence-transformers
5
QA checks & save

Placeholder integrity, HTML tag counts, empty-translation checks. Status is set to machine_draft (or flagged if QA issues found). Reviewers then see these strings in the review queue.

Need to run large batches?

For production-scale runs, use the management command directly — it supports progress logging, chunking, and does not time out: 

python manage.py run_pipeline \
  --locale yo \
  --engine nllb \
  --limit 200 \
  --verbose
